# Calculadora

Projeto de uma calculadora desenvolvida com HTML, CSS e JavaScript.

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## Funcionalidades
- Operações básicas (+, -, *, /)
- Separação de arquivos (HTML, CSS e JS)

## Objetivo
Projeto criado para praticar manipulação de DOM e lógica em JavaScript.
